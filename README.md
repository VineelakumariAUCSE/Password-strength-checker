# Password Strength Checker

## Description
A simple Python-based tool that evaluates password strength using regular expressions and logical rules.

## Features
- Checks password length
- Validates uppercase and lowercase letters
- Validates numbers and special characters
- Classifies strength as Weak, Medium, or Strong

## Technologies Used
- Python
- Regular Expressions (re module)

## How to Run
1. Download or clone the repository
2. Run the file using:
   python password_checker.py
3. Enter a password to see its strength

## Use Case
Helps users create secure passwords and understand password security basics.
